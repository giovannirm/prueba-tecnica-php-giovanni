# Ejecutar Backend
Ejecutar apache

Abrir el proyecto en la terminal

Acceder a la carpeta public

`cd public`

Ejecutar desde la terminal
php -S 127.0.0.1:8080

Entrar al navegador a la ruta

`127.0.0.1:8080`

# Ejecutar los Tests

Desde la ruta principal del proyecto, ejecute el siguiente comando

`composer tests`
